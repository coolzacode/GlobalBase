# GlobalBase

**A minimalist country information dashboard with search functionality.**

> **Note:** Automatically detects the system's theme settings and switches between light and dark modes.

## Key Features

- **Search:** Dynamic, filterable dropdown list to find countries quickly.
- **Asynchronous Data Fetching:** API integration with proper loading states and error handling.
- **Responsive UI:** Fluid design that adapts to various screen sizes.

## How to Run This Locally

1. Clone the repository to your local machine.
2. Navigate into the project directory.
3. Install dependencies: Run `npm install`.
4. Create a `.env` file and add your `COUNTRIES_API_KEY`.
5. Launch the Webpack local development server: Run `npm start`.
6. Open your browser and navigate to the local address displayed in your terminal output.
